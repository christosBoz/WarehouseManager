# Warehouse Management System

A universal warehouse management system for small businesses that tracks inventory movement, monitors stock levels, and manages upcoming jobs with alerts and status visibility.

## Features
- Inventory tracking with adjustments and history
- Job management with priority and status
- Alerts for low stock and upcoming work
- Full CRUD workflows

## Tech Stack
- Frontend: React, TypeScript, Vite, MUI
- Backend: Java, Spring Boot
- Database: SQLite
